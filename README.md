Jal Sanchay💧
Overview
Jal Sanchay is an Android application developed using Kotlin and Jetpack Compose to promote rainwater harvesting and water conservation. The app helps users calculate the amount of rainwater that can be harvested based on roof area, rainfall, and tank capacity.
The project aims to create awareness about sustainable water management through a simple and user-friendly mobile application.
---

Rainwater harvesting calculation
Tank capacity visualization
Monthly water-saving reports
Water conservation tips
Clean and modern UI using Jetpack Compose
Easy user input and result display

Working of the Application
User enters:
Roof area
Rainfall value
Tank capacity
The application calculates the estimated harvested rainwater.
Results are displayed along with:
Water saved in liters
Tank fill progress
Monthly report section
Water-saving suggestions
Users can also access the Guide Screen:
Learn about rainwater harvesting
Understand app usage instructions
View water conservation guidance
---
Formula Used
Water Saved = Roof Area × Rainfall × 0.0929 × Runoff Coefficient
Where:
Roof Area = Catchment area in square feet
Rainfall = Rain received in millimeters
Runoff Coefficient = Efficiency factor of water collection
